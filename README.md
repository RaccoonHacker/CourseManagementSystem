# 智慧教育课程系统 (Smart Education Course System)

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://www.python.org/)
[![Django](https://img.shields.io/badge/Django-3.2%2B-green)](https://www.djangoproject.com/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

本系统是一个面向现代化智慧校园设计的在线选课与教务管理平台。系统通过 **Django** 框架构建，实现了教务流程的数字化重构，并采用了流行的**毛玻璃拟态 (Glassmorphism)** 视觉风格。

---

## 📸 系统演示

### 首页与选课大厅
系统首页采用响应式布局，支持多终端流畅访问。

---

## 🚀 核心功能

- **智能选课模块**：支持全量课程矩阵索引，内置后端冲突校验算法，确保选课逻辑严谨。
- **交互式学术反馈**：师生间即时通讯链路，支持作业提交、在线批改与成绩实时反馈。
- **分布式逻辑讨论社区**：每门课程拥有独立交互空间，促进深度学术对齐。
- **现代化 UI 设计**：基于 CSS3 `backdrop-filter` 技术的毛玻璃拟态界面。

---

## 🛠️ 技术栈

- **后端**: Python 3.x, Django (ORM, Auth, Router)
- **数据库**: MySQL / SQLite
- **前端**: HTML5, CSS3 (Glassmorphism), JavaScript
- **部署**: Gunicorn / Runserver
